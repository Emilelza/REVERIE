🌸 Reverie — Shared Diary for Long Distance

Feel close, even when miles apart.

Reverie is a private shared diary for two people who live far apart — couples, best friends, siblings, or family members.

Instead of fast chat messages, Reverie helps people save memories, emotions, and meaningful moments together.


🚩 Problem

Maintaining emotional connection in long-distance relationships is difficult.

-People fear being forgotten

-Messaging apps focus on quick chats, not memories

-Important moments get lost in conversation history

-No simple tool resurfaces old shared memories


💡 Solution

Reverie provides a shared digital diary where two people can:

-Write daily memories

-React to entries

-Bookmark special moments

-Receive reminders of past memories

The goal is to preserve relationships through memories.


✨ Features
📝 Shared Diary

Write entries and store moments together.

⏳ Memory Reminders

Reverie reminds users of past memories on the same date.

🟢 Together Mode

See when your partner is online.

🌍 Time Zone Display

Shows both users’ local time.

⭐ Favourite Memories

Bookmark special diary entries.

❤️ Reactions

React to entries with emojis.

⚙️ Installation

1️⃣ Clone the repository
git clone https://github.com/Emilelza/shared-diary.git
cd shared-diary

2️⃣ Backend Setup
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Frontend runs on:
-http://localhost:5173

Backend runs on:
-http://127.0.0.1:8000
